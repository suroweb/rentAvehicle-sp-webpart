**Add filter handler** (after the state declarations, before `fetchLocations`):

```typescript
const handleTimezoneInputChange = React.useCallback((text: string): void => {
  setTimezoneSearchText(text);
  if (!text || text.trim() === '') {
    setFilteredTimezones(TIMEZONE_OPTIONS);
    return;
  }
  const query = text.toLowerCase();
  const filtered = TIMEZONE_OPTIONS.filter(
    (opt: ITimezoneOption) => opt.text.toLowerCase().includes(query)
  );
  setFilteredTimezones(filtered);
}, []);
```

**Add menu open handler** to reset search state when ComboBox opens:

```typescript
const handleTimezoneMenuOpen = React.useCallback((): void => {
  setTimezoneSearchText('');
  setFilteredTimezones(TIMEZONE_OPTIONS);
}, []);
```

**Replace the ComboBox JSX** (the editing state block, lines 196-212) with:

```tsx
if (editingLocationId === item.id) {
  return (
    <ComboBox
      allowFreeform={true}
      autoComplete="off"
      options={filteredTimezones}
      text={timezoneSearchText}
      onChange={(_ev: React.FormEvent<IComboBox>, option?: IComboBoxOption) => {
        if (option) {
          handleTimezoneChange(item.id, option.key as string).catch(() => { /* handled in callback */ });
        }
      }}
      onInputValueChange={handleTimezoneInputChange}
      onMenuOpen={handleTimezoneMenuOpen}
      onBlur={() => {
        setEditingLocationId(null);
        setTimezoneSearchText('');
        setFilteredTimezones(TIMEZONE_OPTIONS);
      }}
      openOnKeyboardFocus
      shouldRestoreFocus={false}
      className={styles.timezoneComboBox}
      placeholder="Search timezone..."
      calloutProps={{
        calloutMaxHeight: 320,
        styles: { root: { minWidth: 320 } }
      }}
    />
  );
}
```

Key changes explained:
- `autoComplete="off"` -- disables Fluent UI's inline prefix auto-completion that was confusing users
- `text={timezoneSearchText}` replaces `selectedKey={tz}` -- controls displayed input text directly so it starts empty
- `onInputValueChange` -- filters TIMEZONE_OPTIONS on each keystroke via substring match
- `onMenuOpen` -- resets filter state so full list is available when dropdown first opens
- `options={filteredTimezones}` -- passes filtered array instead of static full array
- `calloutProps` with calloutMaxHeight=320 -- constrains dropdown height for a cleaner appearance
- `shouldRestoreFocus={false}` -- prevents focus jump issues on blur in DetailsList context
- Removed `selectedKey` -- using controlled `text` instead so the input starts blank for fresh search
