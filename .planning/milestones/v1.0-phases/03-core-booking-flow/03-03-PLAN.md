On mount: call `apiService.getMyBookings()`. Categorize bookings by comparing current UTC time against startTime and endTime:

- **Past**: endTime <= now OR status === 'Cancelled' (show both completed and cancelled in past tab)

- Active/Past tabs: `showCancel={false}`

**Cancel flow**: On cancel button click in BookingEntry, show `ConfirmDialog` (reuse existing component from Phase 2) with title "Cancel Booking" and message "Are you sure you want to cancel this booking?" **Empty state** (per locked decision): when all tabs are empty (no bookings at all), show a friendly centered message: "No bookings yet" with a `PrimaryButton` labeled "Book a Vehicle" that calls `onNavigateToBrowse()`. Update `renderPage` switch statement to add cases:

```
case 'browse':
  // If selectedVehicleId is set, show VehicleDetail page
  if (selectedVehicleId !== null) {
    return <VehicleDetail
      vehicleId={selectedVehicleId}
      apiService={apiService}
      onBack={() => setSelectedVehicleId(null)}
      onNavigateToMyBookings={() => { setSelectedVehicleId(null); setActiveNavKey('myBookings'); }}
    />;
  }
  // Otherwise show VehicleBrowse
  return <VehicleBrowse
    apiService={apiService}
    onNavigateToDetail={(id) => setSelectedVehicleId(id)}
    userOfficeLocation={auth.user?.officeLocation || null}
  />;

case 'myBookings':
  return <MyBookings
    apiService={apiService}
    onNavigateToBrowse={() => { setActiveNavKey('browse'); }}
  />;
```
