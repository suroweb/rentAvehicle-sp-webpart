## Tests

### 1. RangeCalendar Visual Rendering
expected: Open any vehicle's Detail page. The BookingForm shows an inline single-month calendar grid (not a popup DatePicker). Clicking a date highlights it with a solid blue circle. Clicking a second date shows circles on both with a light blue band between them. Two hour dropdowns appear below labeled "Start time" and "End time".
result: pass

### 2. Strip Range Overlay and Drag Handles
expected: With a range selected, the Week View strip shows a semi-transparent blue overlay band spanning the selected time range across day columns. Drag handles appear at top/bottom of overlay. Dragging a handle adjusts the range. Overlay disappears when range is outside visible week but reappears when navigating back. Range is never cleared by week navigation.
result: pass

### 3. Day View Range Overlay
expected: Switch to Day View with a range overlapping the selected day. Blue overlay spans correct hour columns across all vehicle rows. Clicking two free slots sets start/end hours. Drag handles snap to boundary when dragged into booked slot. Changes immediately reflect in calendar and hour dropdowns.
result: skipped
reason: Product decision — user wants to remove Day View entirely. Week View per vehicle is sufficient.

### 4. Browse Page Filter Bar Layout
expected: Browse page filter bar shows Location and Category dropdowns in a row, then an inline calendar below (not a DatePicker popup), then Start time and End time hour dropdowns. Click a vehicle card after selecting a range — Detail page shows the same range pre-filled.
result: pass

### 5. End-to-End Booking with Multi-Day Range
expected: Select a range spanning 2+ days, proceed through Review and Confirm. Review panel shows correct multi-day start/end. Booking submits successfully. On mobile, StickyBottomBar shows full multi-day label (e.g. "Feb 25 09:00 - Feb 27 17:00") without overflow.
result: pass

### 6. Single-Day Same-Date Range
expected: Click the same date twice in the RangeCalendar. A single solid circle is displayed (not two circles with a band). Clean single-circle appearance.
result: pass
